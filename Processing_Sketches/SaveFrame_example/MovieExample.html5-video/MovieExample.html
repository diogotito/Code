Nota:


Alguns tipos de software de edição de html podem mudar o código colado segundo os seus standards. Se utilizares este programa, confirma que podes reproduzir o vídeo no teu navegador corretamente e depois sobe-o ao teu sítio web. Se não puderes reproduzi-lo, recomendamos-te utilizar Bloco de notas (Inicio->Todos os programas->Acessorios->Bloco de notas) para verificar o código e corrigi-lo.
